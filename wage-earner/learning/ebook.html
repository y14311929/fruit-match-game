第一章：Python 简介

1.1 什么是 Python？

Python 是一种高级编程语言，由 Guido van Rossum 于 1991 年创建。它以其简洁的语法和强大的功能而闻名，是目前最受欢迎的编程语言之一。

1.2 Python 的特点

  • 简洁易读：Python 语法简洁，代码可读性强
  • 跨平台：支持 Windows、Mac、Linux 等操作系统
  • 丰富的库：拥有庞大的标准库和第三方库
  • 开源免费：完全免费，社区活跃
  • 应用广泛：Web 开发、数据分析、人工智能、自动化办公等
💡 小贴士：Python 的名字来源于英国喜剧团体"蒙提·派森的飞行马戏团"（Monty Python's Flying Circus），而不是蟒蛇。

1.3 Python 的应用领域

  • Web 开发：Django、Flask 等框架
  • 数据分析：Pandas、NumPy、Matplotlib
  • 人工智能：TensorFlow、PyTorch、Scikit-learn
  • 自动化办公：自动处理 Excel、Word、PDF 等
  • 网络爬虫：Scrapy、BeautifulSoup
  • 游戏开发：Pygame 等
⚠️ 注意：Python 有两个主要版本：Python 2 和 Python 3。Python 2 已于 2020 年停止维护，建议学习 Python 3。

1.4 第一个 Python 程序

让我们来看一个简单的 Python 程序：

print("Hello, World!") print("欢迎来到 Python 世界！") # 简单的计算 a = 10 b = 20 print(f"{a} + {b} = {a + b}")

1.5 本章小结

  • Python 是一种简洁、强大的高级编程语言
  • Python 应用广泛，包括 Web 开发、数据分析、人工智能等
  • 建议学习 Python 3 版本
  • Python 代码简洁易读，适合初学者
