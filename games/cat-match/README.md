## 🎯 游戏玩法

### 基本规则
1. 点击一个水果选中（会发光）
2. 点击相邻的水果交换位置
3. 3 个或以上相同水果连成一线即可消除
4. 在步数用完前达到目标分数即可过关

### 连击奖励
- 3 连消：基础分数
- 4 连消：1.3 倍分数
- 5 连消+：1.6 倍分数 + 特效展示

### 道具系统

| 道具 | 价格 | 效果 |
|------|------|------|
| ➕ +5 步 | 50 金币 | 增加 5 步移动次数 |
| 🔄 重置棋盘 | 100 金币 | 重新排列所有猫咪 |
| 💡 提示 | 30 金币 | 显示可消除的位置 |
| 🎲 洗牌 | 80 金币 | 随机打乱整个棋盘 |
